### 1. 登录

- 更新时间：2022年10月12日 20时30分20秒

- 用于获取令牌
- 不要每一次获取号码都访问一次此接口，获得令牌后，后续就不要再次请求


**请求URL **
- ` https://服务器地址/sms/?api=login&user=用户名&pass=密码 `
  
**请求方式**
- GET/POST

**参数**

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|user |  是  |    string   |    用户名   |
|pass |  是  |    string   |    密码   |


**返回成功示例**
``` 
{
    "msg": "success",
    "code": 0,
    "token": "2f05a475cc82f05a4cc82f05a475cc8"
}
```
**返回参数说明 **

|参数名       |说明     |
|:-----       |-----                           |
|code         |状态码，code=0为成功，code=-1为失败  |
|msg          |中文描述|
|token          |令牌|



**备注 **
- code=0或200为成功，code=-1为失败
- 令牌为固定值,除用户修改密码,否则不会变，开发者在写软件的时候登录只请求一次获取到令牌后即可！
